# 农药知识库 — 页面设计

---

## 一、设计原则

| 原则 | 说明 |
|------|------|
| 按钮优先 | 全程点选操作，避免手动输入，降低出错率 |
| 多选支持 | 作物、防治对象均支持多选，取多条件交集 |
| 动态感知 | 每步操作后，下一步可选项实时更新，不出现"0结果"死路 |
| PC 专用 | 仅面向桌面浏览器，固定最小宽度 1280px，不做移动端适配 |
| 零学习成本 | 不需要培训，打开即会用 |

---

## 二、色彩规范

```
主色（深绿）：  #2E7D32   — 导航栏、主要按钮、选中态
辅助色（浅绿）：#E8F5E9   — 选中标签背景
强调色（橙黄）：#F57F17   — 查询按钮、重要提示
背景色：        #F4F6F4   — 页面大背景
卡片背景：      #FFFFFF
文字主色：      #212121
文字次色：      #757575
边框色：        #E0E0E0
```

**标签三态：**

```
未选中：#F5F5F5 背景 + #555 文字 + 1px #DDDDDD 边框
选中：  #E8F5E9 背景 + #2E7D32 文字 + 2px #2E7D32 边框 + ✓ 前缀
禁用：  #FAFAFA 背景 + #BDBDBD 文字，不可点击（该选项下无产品时）
```

**布局说明：**
- 左侧主区域（约 80% 宽）：逐步引导的筛选按钮组 + 底部查询按钮
- 右侧侧边栏（约 20% 宽）：当前已选条件汇总，常驻可见，点击各条件可单独取消
- 结果区：位于筛选区下方，点击查询按钮后显示，卡片两列布局

### 3.2 多选标签交互细节

```
操作逻辑：
  - 单击未选中标签 → 变为选中态（绿底+✓）
  - 单击已选中标签 → 取消选中，恢复默认态
  - 可同时选中多个（OR 逻辑：产品覆盖其中任意一个即匹配）
  - 选中数量 > 0 时，步骤标题旁显示角标：第③步：作物 ②

状态联动：
  - 第③步选择变化时，第④步标签自动刷新（仅显示有对应产品的选项）
  - 第④步选择变化时，查询按钮上的结果计数实时更新
  - 当前条件下无产品覆盖的标签显示为禁用态（灰色不可点击）
  - 若当前条件下无任何产品，查询按钮变灰并提示"当前条件无匹配产品"

"展开/折叠"规则：
  - 标签数量 ≤ 10 个时全部展示
  - > 10 个时展示前 8 个 + [展开全部 N 个 ▼] 按钮
  - 点击展开后按钮变为 [收起 ▲]
```

### 3.3 面包屑导航

```
位于结果区顶部，记录当前选择路径：

  除草 › 夏季除草剂单剂 › 花生、大豆 › 马唐、稗草

规则：
  - 各段均可点击，点击后回退到该步并清除后续所有选择
  - 多选时用顿号连接（如"花生、大豆"）
  - 超过 3 个选项时显示"花生等 3 种"
```

### 3.4 产品结果列表

结果区采用**多列宽行表格**布局，每行一个产品，列定义如下：

| 列名 | 宽度占比 | 说明 |
|------|---------|------|
| 产品名称 | 25% | 含量+名称+剂型，粗体显示 |
| 用量 | 10% | 每桶用量，居中 |
| 适用作物 | 30% | 该产品关联的所有作物，换行平铺显示 |
| 防治对象 | 30% | 该产品关联的所有防治对象，换行平铺显示 |
| 操作 | 5% | [详情] 按钮 |

**高亮规则（核心交互）：**

- 用户在筛选步骤中已选中的作物，在"适用作物"列中该标签显示为**绿色背景+深绿文字**（与筛选标签选中态一致）
- 用户已选中的防治对象，在"防治对象"列中同样**绿色背景高亮**
- 未被用户选中的作物/防治对象，保持普通浅灰标签样式
- 直接搜索模式下无高亮（无明确选中项）

### 3.5 直接搜索模式

顶部搜索框为备用快速入口（熟悉产品的用户可跳过步骤直接搜索）：

```

- 搜索范围：产品名称 + 有效成分 + 防治对象名称
```

## 五、关键交互规则汇总

| 规则 | 说明 |
|------|------|
| 多选逻辑（作物） | OR：选花生+大豆，返回覆盖花生或大豆的产品 |
| 多选逻辑（防治对象） | OR：选马唐+稗草，返回能防治马唐或稗草的产品 |
| 跨步骤逻辑 | AND：作物条件与防治对象条件取交集 |
| 动态禁用 | 当前条件下无产品覆盖的选项自动变灰，不可点击 |
| 步骤回退 | 点击面包屑任意节点，清除该节点之后的选择 |
| 右侧已选摘要 | 常驻显示当前已选条件，点击各条件可单独取消 |
| 结果计数 | 查询按钮实时显示当前条件下的产品数量 |
| 内联新增 | 管理弹窗内可直接新增作物/防治对象，无需跳转页面 |
| 删除保护 | 已被产品引用的作物/防治对象/分类不能直接删除 |
| 下架隐藏 | 下架产品查询页不可见，管理页显示并以灰色标注 |
